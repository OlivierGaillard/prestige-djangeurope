Comment saisir une vente

  1. recherchez l'article que vous avez vendu en cliquant sur le lien "liste des articles"
  2. cliquez sur le lien du "nom" de l'article
  3. ajoutez l'article au panier
  4. entrez le prix de vente
  5. si vous avez vendu d'autres articles, cherchez les et ajoutez les au panier
  6. une fois que votre panier est rempli, vérifiez si vous avez besoin de saisir un nouveau client (le rajouter ou le sélectionner dans la liste), ou bien si votre vente est anonyme
  7. passez la commande, puis sélectionnez le client si besoin (les tirets indiquent une vente anonyme)
  8. modifiez la date si nécessaire
  9. appuyez sur le bouton "submit" pour enregistrer.
  10. cliquez sur ajouter le paiement du client
  11. modifiez le prix
  12. cliquez sur "submit" pour terminer la vente

Ajouter un paiement

Lorsque le client paie par avances, vous devez afficher la liste des ventes, puis cliquer sur le vente-ID afin d'ajouter le paiement.
